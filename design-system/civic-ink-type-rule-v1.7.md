# Civic Ink v1.7 — Typographic roles (addendum)

Decided 2026-07-16 while designing the Field Notes homepage. Structure typeface updated to Schibsted Grotesk in v1.7 (2026-07-22); the rule itself is unchanged.

## The rule

- **Schibsted Grotesk is structure.** Mastheads, nav, section eyebrows, labels, dates, deks, buttons, footers — anything that would still exist if the content changed.
- **Playfair Display is voice.** It appears only where I am saying something: theses, entry titles, pull quotes, prose emphasis. Serif is a scarcity signal; every appearance carries editorial weight because it is rationed.
- **A toy's display face (e.g., Syne for find-a-film) is that toy's voice**, and stays inside the toy's own surfaces. Toy chrome remains Schibsted Grotesk.

The test for any new element: *is this chrome, or is this me talking?* The answer sets the typeface. No further debate required.

## Sizing

- Headers sit on multiples of 8: **16 / 24 / 32** (40 in reserve).
- Body text and small labels align to the 8pt grid via **line-height**, not font size (e.g., 13px label / 16px leading is compliant).

## Why

Before this rule, Playfair did two jobs — brand chrome (wordmark) and voice (thesis, titles) — so typeface carried no signal and hierarchy had to be argued through size alone (the 22px wordmark vs. 24px headline standoff). Splitting the roles resolves hierarchy structurally and gives a durable decision rule instead of a per-page judgment call.

## Rejected

- Keeping the Playfair wordmark and settling the conflict by size/weight nudges — works once, doesn't generalize.
- Strict multiples of 8 for *all* font sizes — leaves nothing usable between 8 and 16; the grid is enforced through leading below header scale.
